== Description ==

GameInfo Terminal is a unique WordPress theme designed for gaming news websites.
It features a terminal/console aesthetic with monospace typography, animated cursors,
and a dark hacker-inspired design.

Features:
* Dark console-style design
* Terminal-inspired typography with JetBrains Mono
* Animated cursor effects
* Category tabs navigation
* AJAX load more posts
* Custom post meta fields for source attribution
* Responsive design
* Translation ready

== Installation ==

1. Upload the theme folder to /wp-content/themes/
2. Activate the theme through the Themes menu in WordPress
3. Configure theme options in Appearance > Customize

== Menu Locations ==

* Primary Menu - Main navigation in the header (displayed as [menu-item])
* Category Tabs - Horizontal category navigation below page title
* Footer Menu - Bottom links (EULA, Privacy, etc.)

== Customizer Options ==

* Terminal Title - The site title displayed in terminal format
* Terminal Path - The path shown next to the title (e.g., ~/news)
* Build Version - Version number shown in footer

== Post Features ==

Each post can have a custom "Source" field that displays as the source tag.
Categories and tags with "leak" or "critical" in the name will display with
special styling.
